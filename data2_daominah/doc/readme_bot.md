# Guide References

## .ai file

Should edit in dir data2_daominah then copy to files in data2 with the
corresponding civilization.

Example 1 file `data2_daominah/Hittite_Horse_Archers.ai` is copied to 3 following files in dir `data2`:  

- `Hittite Bowmen.ai`
- `Hittite Elephant.ai`
- `Hittite Horse Archers.ai`

All the Hittite computers play the same way, we can add more build later.

[ai_file_build_order.pdf](./ai_file_build_order.pdf)
https://aoe.heavengames.com/siegeworkshop/ai/

## .per file

(not important because cannot use in Random Map)

PER and CTY files listed in the data and data2 folders can
__only be used for your custom-made scenarios and campaigns__.
In multiplayer Random Map or Death Match the computer will use default
PER and CTY files, they are hard coded into the game exe.
